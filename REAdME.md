
# Erykah Wanga Portfolio

A professional, "Sakura-themed" portfolio for a Software Engineer.

## Local Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **AI Functionality**:
   To use the AI assistant locally, open your browser console (F12) and type:
   ```javascript
   window.LOCAL_API_KEY = 'your_gemini_api_key_here'
   ```
   Get a free key at [ai.google.dev](https://ai.google.dev/).

## Technologies
- React 19
- Tailwind CSS
- Lucide React
- Google Gemini API (GenAI SDK)
